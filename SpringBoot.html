SpringBoot配置

两种配置文件类型 properties 和yml，其中properties>yml>yaml

YAML

基本语法

是以数据为中心的，比传统xml更简洁，使用缩进表示层级关系，缩进只能使用空格，不可以用tab，缩进的空格数据不重要，同层级元素左对齐。

数据格式

  1. 对象（map)

  2. 数组

  3. 纯量–常量 单引号不识别转义

  4. 参数引用 ${name}

读取配置文件内容

@Value(”${key}“); Environment.getProperty(“key”) + @Autowired ; @ConfigrationProperties(prefix=”person”)

Profile

  • profile 切换配置

    1. 多profile文件方式
    2. yml多文档方式

  • profile激活方式

    1. spring.profiles.active=dev激活配置文件
    2. VM options： -Dspring.profiles.active=dev
    3. Program arguments: --spring.profiles.active=dev

配置文件加载顺序

内部配置文件加载顺序：file:/config/ > file:/ > classpath:/config/ > classpath:/

修改访问路径：server.servlet.context-path

外部配置文件加载顺序： config文件夹>target根目录配置>内部配置
